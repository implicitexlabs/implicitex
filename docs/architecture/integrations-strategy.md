# Integrations Strategy

## Ledger Hardware Wallet Integration
- Ledger integration is planned but deferred.
- Prerequisites:
  - Stable web presence (production-ready online tool)
  - Stable Electron desktop software
- Implementation starts only after both prerequisites are met.

## AuditWalk Security Complement
- AuditWalk will be used as a complementary security tool to ImplicitEx.
- AuditWalk scope in this context:
  - Security validation support
  - Risk visibility and trust signaling support
- AuditWalk complements ImplicitEx product delivery and does not replace core transaction functionality.
